HTML:


Hypertext Markup Language, es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que se utiliza para estructurar el contenido de una página web, definiendo elementos como texto, imágenes, enlaces, formularios y otros elementos multimedia. HTML utiliza etiquetas (tags) para definir la estructura y el significado del contenido dentro de una página web.

BODY:


Es una etiqueta que se utiliza para definir el cuerpo principal de una página web. Todo el contenido visible de la página web, como texto, imágenes, enlaces, formularios, etc., se coloca dentro de esta etiqueta.

CENTER:


es un término que se refiere al proceso de alinear elementos en el centro de un contenedor o de la pantalla. Esto puede aplicarse a texto, imágenes, bloques de contenido y otros elementos dentro de una página web.

H1:


Esta etiqueta define un encabezado de nivel 1, que generalmente se muestra con un tamaño de fuente más grande que el texto normal y a menudo se utiliza para títulos principales.

HR:


Esta etiqueta se utiliza para insertar una línea horizontal en la página, que se puede utilizar para separar secciones de contenido.

BR:


Se utiliza para insertar un salto de línea o un quiebre de línea dentro de un texto.Cuando el navegador encuentra la etiqueta br>9, crea una ruptura en el flujo del texto y coloca el texto siguiente en una nueva línea, similar a presionar la tecla "Enter" en un procesador de texto.

P:


Esta etiqueta define un párrafo de texto.

HEAD:


provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

TITLE:


HTML define el título del documento que se muestra en un browser la barra de título o la pestaña de una página. Solo contiene texto; las etiquetas dentro del elemento se ignoran. Contenido de metadatos. Texto que no es un espacio en blanco entre elementos.
